Datenschutzerklärung

WonderWalk · Stand: Mai 2025

WonderWalk ist ein GPS-basierter KI-Audioguide für Android. Diese Datenschutzerklärung erläutert, welche Daten die App überträgt, an wen und zu welchem Zweck.

Grundsätze

  • Es werden keine persönlichen Nutzeridentifikatoren erhoben oder gespeichert.
  • Die App betreibt keine eigenen Server. Alle Daten gehen direkt an öffentliche Drittdienste.
  • Standortdaten werden nur als Teil von Such- und Geocodieranfragen übertragen – nie dauerhaft auf einem Server gespeichert.
  • Alle lokal gespeicherten Daten (API-Keys, Verlauf, Cache) verbleiben ausschließlich auf dem Gerät.
  • Es gibt keine Werbung, kein Tracking und keine Analyse des Nutzerverhaltens.

Datenübertragungen

Pflicht Google Gemini API (api.gemini.google.com) Wenn du eine KI-Anfrage stellst oder der Auto-Modus aktiv ist, überträgt die App den aufgebauten Prompt an die Gemini API. Dieser kann enthalten: POI-Namen, OSM-Tags, Wikipedia-Text, Wikidata-Auszüge sowie optional ein Foto des POIs oder ein selbst aufgenommenes Kamerabild.

Der API-Key wird lokal auf deinem Gerät gespeichert und nicht an Dritte weitergegeben. Es gelten die Google Gemini API Terms of Service und die Google Datenschutzrichtlinie.
Pflicht OpenStreetMap Overpass API (overpass-api.de) Zur Suche nach Points of Interest in der Nähe werden GPS-Koordinaten und der gewählte Suchradius an die öffentliche Overpass API übertragen. Es werden keine persönlichen Daten übermittelt. Die Nutzung unterliegt den OSMF Usage Policies.
Pflicht Nominatim Geocodierung (nominatim.openstreetmap.org) Zur Umwandlung von GPS-Koordinaten in eine lesbare Adresse (Reverse Geocoding) werden Koordinaten an Nominatim übertragen. Es werden keine persönlichen Daten übermittelt. Nutzungsrichtlinien: OSMF Nominatim Policy.
Pflicht Wikipedia / Wikidata / Wikimedia Commons Zum Abrufen von Artikeln, Fakten und Medien zu einem POI werden Anfragen an die öffentlichen APIs von Wikipedia, Wikidata und Wikimedia Commons gestellt. Übertragen werden ausschließlich die Bezeichnung oder ID des betreffenden Objekts sowie GPS- Koordinaten für Geo-Suchen. Es werden keine persönlichen Daten übermittelt.
Pflicht OpenHistoricalMap API (openhistoricalmap.org) Wenn die OHM-Datenquelle aktiviert ist, werden GPS-Koordinaten zur Suche nach historischen POIs an die OHM Overpass API übertragen. Es werden keine persönlichen Daten übermittelt.
Optional Wikimedia Commons Upload (commons.wikimedia.org) Wenn du Fotos oder Tonaufnahmen zu Wikimedia Commons hochladen möchtest, wird ein Wikimedia-OAuth2-Token benötigt. Dieses Token wird ausschließlich lokal auf deinem Gerät gespeichert und nicht an Dritte weitergegeben. Beim Upload werden die Mediendatei sowie von dir eingegebene oder KI-generierte Metadaten (Beschreibung, Kategorien) an Wikimedia Commons übermittelt. Hochgeladene Inhalte werden unter einer freien Lizenz (CC BY-SA 4.0) veröffentlicht.
Optional Google Play Billing (play.google.com) Für den Kauf von WonderWalk Pro wird die Google Play Billing API genutzt. Der Kaufvorgang wird vollständig über Google Play abgewickelt. WonderWalk speichert keine Zahlungsdaten. Es gelten die Google Play Nutzungsbedingungen.
Pflicht GitHub API (api.github.com) Beim App-Start prüft WonderWalk einmalig über die öffentliche GitHub API, ob eine neue Version verfügbar ist. Es werden keine persönlichen Daten übermittelt.

Lokal gespeicherte Daten

Folgende Daten werden ausschließlich auf dem Gerät gespeichert und verlassen es nicht:

  • Gemini API-Keys – in den App-Einstellungen gespeichert
  • Wikimedia OAuth-Token – nur wenn du dich eingeloggt hast
  • KI-Antwort-Cache – Gemini-Antworten, 30 Tage, löschbar unter Einstellungen → Cache
  • Anreicherungs-Cache – Wikipedia-Texte, Bilder, Wikidata-Auszüge, 7 Tage
  • Ambiente-Klang-Cache – heruntergeladene Audiodateien, 30 Tage, max. 25 Dateien
  • POI-Verlauf – bereits besuchte und angekündigte POIs
  • UI-Übersetzungen – lokal generierte Übersetzungen der App-Oberfläche
  • App-Einstellungen – Radius, Prompt-Vorlagen, Filter, Favoriten

All diese Daten können unter Einstellungen → Cache leeren oder über die Android-Systemeinstellungen vollständig gelöscht werden.

Berechtigungen

  • Standort (Präzise & Ungefähr) – für die POI-Suche und den Auto-Modus; nur während der Nutzung aktiv, im Auto-Modus als Foreground Service
  • Kamera – optional, nur wenn du ein Foto aufnehmen oder hochladen möchtest
  • Mikrofon – optional, nur für Tonaufnahmen zum Upload auf Wikimedia Commons
  • Benachrichtigungen – für die Statusleiste des Foreground Service im Auto-Modus
  • Internet – für alle API-Anfragen
  • Externer Speicher (Android ≤ 12) – für den RSS-Export in den Downloads-Ordner

Kinder

WonderWalk richtet sich nicht gezielt an Kinder unter 13 Jahren. Die App erhebt keine personenbezogenen Daten und ist nach IARC als FSK 0 eingestuft.
